# flophouse 
page with register form for adaptive site of non-profit organization homeless.ru (https://www.facebook.com/Homeless.ru). 
Each step of the form rendered according to the previous user choice.

Used technologies: 
- JS
- CSS (BEM)
- HTML
- Git
